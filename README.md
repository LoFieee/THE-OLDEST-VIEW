# THE OLDEST VIEW

A Minecraft Fabric analog horror experience inspired by Kane Pixels' **The Oldest View**.

## About

**THE OLDEST VIEW** is a fan-made Minecraft project that transforms the Backrooms concept into a cinematic exploration experience.

Built from the foundation of **SPB Revamped**, the project aims to introduce:
- Custom environments
- New events and encounters
- Atmospheric gameplay
- Cinematic sequences
- Unique horror elements

## Status

🚧 Work in progress

The project is currently focused on redesigning the experience, replacing original assets, creating a custom title screen, and expanding gameplay systems.
